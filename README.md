# 📄 AI-Powered PDF Chatbot

An intelligent **PDF Question Answering System** that enables users to interact with PDF documents using natural language. Built with **Retrieval-Augmented Generation (RAG)**, **Sentence Transformers**, **OCR**, and a **locally hosted Mistral LLM (via Ollama)**, the chatbot delivers accurate, context-aware answers by understanding the content of uploaded PDFs.

## ✨ Features
- 📂 Upload and analyze PDF documents
- 💬 Ask questions in natural language
- 🔍 Retrieval-Augmented Generation (RAG) for accurate responses
- 🧠 Semantic search using Sentence Transformers
- 🖼️ OCR support for extracting text from images inside PDFs
- 🤖 Locally hosted Mistral LLM using Ollama
- ⚡ Fast and user-friendly Flask web interface

## 🛠️ Tech Stack
- Python
- Flask
- Ollama (Mistral 7B)
- Sentence Transformers
- PyMuPDF
- OCR (Tesseract)
- FAISS / Vector Embeddings
- HTML, CSS, JavaScript

## 🚀 How It Works
1. Upload a PDF document.
2. Extract text and perform OCR on embedded images.
3. Generate semantic embeddings for document chunks.
4. Retrieve the most relevant content using RAG.
5. Generate accurate, context-aware answers with Mistral LLM.
